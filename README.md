# ChatRLB
 AI-Powered Educational Chat Platform  (Sponsored Nonprofit Project)

This project was developed in collaboration with Red Letter Basics, a nonprofit organization focused on providing accessible, foundational Bible education centered on the teachings of Jesus.

Our role as developers was to translate the sponsor’s educational mission into a functional, user-friendly software solution that met defined technical and ethical requirements.

# My Role and Responsibilities 
I contributed as a Front-End Developer and Integration Support Engineer, with responsibilities including:

- Designing and implementing 10+ reusable UI components using React (chat bubbles, sidebars, responsive layouts)
- Collaborating with backend developers to connect front-end features to Django-based APIs
- Supporting and testing AI model integration to ensure stable real-time chat functionality
- Leading development of accessibility features, including Text-to-Speech and Speech-to-Text
- Participating in Agile workflows, sprint planning, and Git-based collaboration with a cross-functional team

# Key Features
- Real-time AI-powered chat interface
- Responsive and accessible UI design
- Text-to-Speech and Speech-to-Text functionality
- Secure communication with backend services
- Modular component-based front-end architecture
- Cross-browser and mobile-friendly support

# Challenges and Lessons
**Challenges**
- Ensuring reliable real-time communication between the frontend and AI backend
- Balancing accessibility features with performance constraints
- Translating non-technical sponsor goals into concrete technical requirements

**Learnings**
- Building production-ready UI components for real users
- Collaborating with stakeholders outside of engineering
- Testing and validating AI-powered features in a live environment
- Designing with accessibility as a core requirement, not an afterthought

## Code Availability

This repository is intended to demonstrate the project’s scope, features, and my individual contributions without exposing proprietary code.
